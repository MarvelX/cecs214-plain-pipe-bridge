# CECS 214-2006 平管管桥计算软件

本项目实现了一个基于 Streamlit 的本地网页应用，用于按 `CECS 214:2006` 对平管管桥进行作用计算、组合验算、内力分析、支墩基础校核和计算书导出。

## 功能范围

- 平管截面参数自动计算
- 永久作用、可变作用和混合作用组合
- 三种支承形式的平管内力与挠度计算
- 强度、真空稳定和支墩基础验算
- 输入 JSON 导入导出
- 中文 HTML 计算书导出

## 运行

```bash
python3 -m pip install -e .[dev]
streamlit run app.py
```

## 测试

```bash
python3 -m pytest
```

## 说明

- 首版仅覆盖平管，不覆盖折管、拱管和附录系数表。
- 风荷载、地基承载力、基础尺寸等外部规范参数由用户手工输入。
- 计算书导出为 HTML，可直接在浏览器中打印为 PDF。
